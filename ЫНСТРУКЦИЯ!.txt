# Версия 2.0
ЫНСТРУКЦИЯ для работы с моим детищем - великолепнейшим скриптом QURT!

1. Сначала экспортируешь реестр учеников из Контингента. Полученный файл "Контингент.xlsx" помещаешь в папку "qurt".
2. Затем запускаешь скрипт "1-students-export-all-students-in-one-csv.ps1". В папке появится файл "exported.csv". Это список учеников из ActiveDirectory.
3. Далее "2-qurt.py" запускаешь, эта дрянь формирует отчет "STUDENTS_DETAILED.xlsx" со списком новеньких, переведенных между классами и выбывшими (если таковые есть). Плюс скрипт предложит сохранить этих учеников в отдельные csv-файлы.
4. При наличии csv-файла с новенькими учениками, надо запустить "3-students-new-accounts.ps1".
5. Если есть ученики, которые перевелись между классами, то открываем "4-students-change-class.ps1".
6. По выбывшим ученикам пока ничего не сделано, придется ручками их перемещать.
